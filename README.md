# Calculator-Roman-Numerals

Result
--
Roman numerals, followed by an operation, followed by Roman numerals. Each should be separated by a space.

Input from text file :
Example
  II + I
  V  +  V
  II  +  II
  II  +  I
  VI + X
  XXX  +  X
  CM + XII
  
Output :
    > III
    > X
    > IV
    > XVI
    > XL
    > CMXII
